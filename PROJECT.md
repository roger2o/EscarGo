# EscarGo!

A mobile-first PWA puzzle game — a "fast snail" leaves a slime trail you can't cross (Snake meets Tron). Built with Phaser, TypeScript, and Vite; deployed as a PWA.

## Retired — 2026-05-16

**Why retired:** Solo hobby project that stalled after two commits in February 2026. Bigger active priorities and the game's UI was not good enough nor worth investing time into.

**Final state:** Prototype only — core movement and collection mechanic shipped with PWA installability, but only one level exists and none of the planned mechanics (gates, teleporters, ice, doors, multi-snail) ever made it in.

**Worth salvaging:** Nothing.

**Live services torn down:** Netlify deployment deleted on 2026-05-16 (site `b32a09fe-b8fb-4ddd-a844-34da752ddbcd`).

**Preserved at:** github.com/roger2o/EscarGo (master in sync, fully pushed).

**Cross-project ties:** None outside `portfolio-status.md`, which regenerates automatically.
